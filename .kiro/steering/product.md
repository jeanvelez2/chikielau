# Product Overview

## Chikielau Website Templates

Three distinct, professional website template designs for the Chikielau bookstagrammer brand. Each template is a complete, independent package ready for immediate deployment on Hostinger.

### Templates

1. **Literary Lounge**: Magazine-style layout with warm cream tones, traditional horizontal navigation, featured book carousel, and "Currently Reading" spotlight
2. **Celestial Bookshelf**: Bold asymmetric design with dramatic black and gold contrast, side/hamburger navigation with celestial icons, and star-rating system
3. **Moonlit Pages**: Clean single-column storytelling layout with soft neutral palette, sticky minimalist navigation, and "Book of the Month" section

### Core Features

- Six essential pages: Homepage, Blog Archive, Single Blog Post, About, Shop/Recommendations, Contact
- Newsletter signup integration (Mailchimp/ConvertKit compatible)
- Instagram feed integration areas
- Affiliate link support for book recommendations
- Mobile-first responsive design (320px to 1920px)
- WCAG 2.1 AA accessibility compliance
- SEO-optimized structure

### Brand Identity

**Chikielau Brand**: Cozy vintage library vibes meets celestial elegance

**Visual Elements**:
- Gold celestial logo with crescent moon and stars
- Elegant script typography
- Color palette: metallic gold, deep black, cream/ivory, warm neutrals

### Target Users

- Bookstagrammers building their online presence
- Content creators showcasing book reviews
- Brand owners seeking professional portfolio sites
- Non-technical users needing easy customization
