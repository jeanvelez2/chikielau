# Chikielau Website Templates

Three distinct, professional website template designs for the Chikielau bookstagrammer brand. Each template is a complete, independent package ready for immediate deployment on Hostinger.

## Overview

This project provides three unique website templates, each embodying the Chikielau brand identity of "cozy vintage library vibes meets celestial elegance" while offering distinct aesthetic interpretations:

- **Template 1: Literary Lounge** - Traditional, magazine-inspired with warm cream tones
- **Template 2: Celestial Bookshelf** - Bold, artistic with dramatic black and gold contrast
- **Template 3: Moonlit Pages** - Minimalist, storytelling-focused with soft neutral palette

## Templates

### Template 1: Literary Lounge

**Design Philosophy**: Magazine-style layout emphasizing cozy elegance and traditional book culture.

**Key Features**:
- Large header with traditional horizontal navigation

## Quick Start

### 1. Choose Your Template

Browse the three templates and select the one that best fits your brand aesthetic:
- `template-1-literary-lounge/` - Traditional magazine style
- `template-2-celestial-bookshelf/` - Bold artistic style
- `template-3-moonlit-pages/` - Minimalist storytelling style

### 2. Customize Your Template

Each template includes a comprehensive README with customization instructions:

1. **Update Logo**: Replace `assets/images/logo.png` with your logo (200x200px recommended)
2. **Modify Colors**: Edit `css/variables.css` to change brand colors
3. **Add Content**: Replace placeholder text and images with your content
4. **Configure Forms**: Connect newsletter and contact forms to your email service
5. **Add Social Links**: Update social media URLs in header/footer

### 3. Deploy to Hostinger

**Option A: File Manager Upload**
1. Log in to your Hostinger account
2. Navigate to File Manager
3. Upload all template files to your `public_html` directory
4. Your site is live!

**Option B: FTP Upload**
1. Use an FTP client (FileZilla recommended)
2. Connect to your Hostinger account
3. Upload all template files to `public_html`
4. Access your site via your domain

### 4. Connect Services

- **Newsletter**: Follow instructions in template README for Mailchimp/ConvertKit integration
- **Instagram Feed**: Add embed code from SnapWidget, Elfsight, or Instagram
- **Contact Form**: Configure form backend (instructions in template README)
- **Affiliate Links**: Add your affiliate URLs to product cards in `shop.html`

## Template Structure

Each template follows an identical directory structure:

```
template-name/
├── index.html                 # Homepage
├── blog.html                  # Blog archive page
├── blog-post.html             # Single post template
├── about.html                 # About page
├── shop.html                  # Shop/recommendations page
├── contact.html               # Contact page
├── css/
│   ├── variables.css         # CSS custom properties
│   └── styles.css            # Main stylesheet
├── js/
│   └── script.js             # Main JavaScript
├── assets/
│   ├── images/
│   │   ├── logo.svg          # Chikielau logo
│   │   ├── placeholders/     # Example images
│   │   └── icons/            # Social media icons
│   └── fonts/                # Web fonts (if self-hosted)
└── README.md                 # Template documentation
```
